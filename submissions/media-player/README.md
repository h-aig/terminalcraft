Hi! Thanks for checking out my project!

### How to run:
- Install dependencies with `pip install -r requirements.txt`
- Run the program:
    - Windows: Run `play [file]`.
    - Mac/Linux: Run `./play.sh [file]`.
- Enjoy!

*Use `play --help` for additional options*
